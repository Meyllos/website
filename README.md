# Meyllos

Website for MeyllOS Software friendly Company
Project in `Angular version 7.3.8.`

## Install TailwinCSS
The frontend part is developed with the `TailwinCss` css framework

## Send a Nofication
Angular Toastr Notifications
Install ```ngx-toastr```
install ```@angular/animations```
